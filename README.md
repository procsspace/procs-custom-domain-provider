# Procs Custom Domain Provider

The subdomain provider for [procs.space](https://procs.space/u/procs)



## Reference

#### Redirect

```http
  GET /*
```

| Parameter | Type     | Description                |
| :-------- | :------- | :------------------------- |
| `username` | `string` | **Required**. The username you would like to redirect to. |
